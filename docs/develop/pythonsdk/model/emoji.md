# 表情对象

用来表示一个表情基本单位。

### Emoji

| 字段名 | 类型   | 描述                                                                                                   |
| ------ | ------ | ------------------------------------------------------------------------------------------------------ |
| id     | string | 表情 ID，系统表情使用数字为 ID，emoji 使用 emoji 本身为 id，参考 Emoji 列表 |
| type   | uint32 | 表情类型 EmojiType                                                          |

### EmojiType

| 值  | 描述       |
| --- | ---------- |
| 1   | 系统表情   |
| 2   | emoji 表情 |
